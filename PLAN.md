# 凯戈一人公司自媒体创业计划

## Context

**凯戈身份：** Magikid AIRobotics Lab CTO & 创始人，硅谷教育科技背景，34 个加盟店，品牌包括 MagiCode / Magikid。

**创业方向：** AI教育 & 少儿科技自媒体，延伸主业品牌，同时打造个人 IP。

**平台：** 全平台 — 微信公众号、小红书、抖音/视频号、YouTube（中英双语）

**分工：**
- 凯戈：出镜拍视频、直播课
- Claude（AI CTO）：内容生产 / 选题策略 / 品牌视觉 / 产品搭建

## 二、内容工厂流水线

```
[选题] → [脚本] → [拍摄] → [剪辑] → [多平台分发]
  ↑Claude   ↑Claude   ↑凯戈    ↑凯戈+AI   ↑Claude
```

**一稿多发原则：** 一个核心选题产出：
1. 长视频脚本（YouTube 10-15min / 抖音 3-5min）
2. 公众号深度文章（1500-3000字）
3. 小红书图文（标题 + 6-9张图 + 文案）
4. 视频号/抖音短视频脚本（60-90秒精华版）
5. YouTube 英文版（翻译 + HeyGen 口型同步）
